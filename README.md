# 🌍 Text Translator App

A simple and interactive **Text Translator Web Application** built using **React**.
This app allows users to enter English text and translate it into multiple languages using an external translation API.

---

## 🚀 Features

* Translate English text into multiple languages
* Clean and simple user interface
* Supports several languages such as:

  * Hindi
  * Tamil
  * Spanish
  * French
  * German
  * Japanese
  * Kannada
  * Telugu
  * Malayalam
  * Italian
  * Portuguese
  * Russian
  * Chinese
  * Korean
  * Arabic
  * Bengali
  * Urdu
  * Indonesian
  * Turkish
  * Dutch
* Real-time API-based translation
* Built with modern frontend tools

---

## 🛠️ Technologies Used

* **React**
* **Vite**
* **JavaScript**
* **RapidAPI (Google Translate API)**

## ⚙️ Installation and Setup

Follow these steps to run the project locally.

### 1. Clone the Repository

```
git clone https://github.com/deepikaselvaraj5106-coder/translator-app.git
```

### 2. Navigate to the Project Folder

```
cd translator-app
```

### 3. Install Dependencies

```
npm install
```

### 4. Start the Development Server

```
npm run dev
```

### 5. Open in Browser

```
http://localhost:5173
```

---

## 🔑 API Setup

This project uses the **Google Translate API via RapidAPI**.

1. Create an account on RapidAPI
2. Subscribe to the Translate API
3. Copy your API key
4. Replace the API key in `App.jsx`:

```
X-RapidAPI-Key: YOUR_API_KEY
```
